浏览器兼容性测试报告

测试环境
  • Chrome 120+、Edge 120+、Safari 16+、Firefox 120+
  • macOS、Windows 11、iOS 17、Android 14
关键样式支持
  • backdrop-filter：Chrome/Edge/Safari 原生支持；Firefox 需启用实验特性，默认走降级
  • @supports 条件：所有现代浏览器支持，用于优雅降级
  • CSS 变量：现代浏览器全面支持，用于统一半径、过渡、模糊等参数
降级与兼容策略
  • 不支持 backdrop-filter 时：使用更不透明的线性渐变替代，并保持统一圆角与边框
  • 阴影强度适配：移动端使用较轻阴影，减少重绘开销
  • 响应式布局：通过 Bootstrap 栅格和自定义 .glass-grid 控制间距与断点
结论

统一液态玻璃样式在现代浏览器表现一致；在不支持模糊的环境下提供稳定的视觉降级，保证可用性与性能。
