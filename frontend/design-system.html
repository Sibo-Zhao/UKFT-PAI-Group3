Design System Overview

Visual Style

• Uses basic frosted glass effect (blur + transparency).

• Rounded corners for softer look.

• Light shadows for simple depth.

Layout Rules

• Standard spacing with Bootstrap grid.

• Titles use .glass-title.

• Keep padding consistent across components.

Interaction

• Hover lifts elements slightly.

• Click returns them to normal state.

• Added simple transitions.
